Tecnologias

Confira as tecnologias que mais utilizo

Java

Ambiente computacional que permite desenvolver programas utilizando a linguagem de programação Java.

Spring Boot

Ferramenta que visa facilitar o processo de configuração e publicação de aplicações que utilizem o ecossistema Spring.

Apache Maven

Ferramenta utilizada para construção e gerenciamento de projetos, permite desenvolvimento de forma padronizada, agregando agilidade e qualidade.

Spring Batch

O Spring Batch é um framework para processamento em lote. Solução leve e abrangente, projetada para permitir o desenvolvimento de aplicativos robustos.

Hibernate

Hibernate é o framework para persistência de dados mais utilizado em projetos Java. Sendo uma das primeiras opções a implementar o conceito de mapeamento objeto-relacional (ORM).

Cucumber

Usado para descrever o valor do negócio em uma linguagem natural. Permite que equipes de desenvolvimento descrevam como o software deve se comportar em texto simples.

MySQL

Sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface.

MongoDB

Software de banco de dados orientado a documentos, de código aberto e multiplataforma. Classificado como NoSQL, usa documentos semelhantes a JSON com esquemas.

Github

Plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Permite que programadores contribuam em projetos de qualquer lugar do mundo.

Docker

Plataforma como serviço que usam virtualização de nível de sistema operacional para entregar software em pacotes chamados contêineres.

Postman

Linguagem de Consulta Estruturada(SQL) é a linguagem de pesquisa declarativa padrão para banco de dados relacional.

MongoDB

NoSQL é um termo genérico que representa os bancos de dados não relacionais.
